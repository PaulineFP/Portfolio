A propos du projet

Le projet « Répare PC » est un projet pour OnlineFormapro. Le but est de créer une application « one page » permettant la prise de rendez-vous dans le but de réparer son ordinateur. Il doit intégrer une carte de France interactive, géolocalisant les différents sites de réparation.
