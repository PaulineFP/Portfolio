A propos du projet

Lors de la formation de designer web nous devions créer une application utilisant un CMS. Pour cela j’ai choisi d’utiliser le CMS Wordpress. J’ai décidé que mon travail porterait sur le bien être sous forme d’un blog. J’ai dû réfléchir sur la cohérence graphique en m’inspirant de modèles déjà existant, trouver les bonnes couleurs, police d’écriture, qui aspirent le bien être. Après avoir trouvé l’inspiration, j’ai conçu la charte graphique, le wireframe et le maquettage. Par la suite l’intégration HTML et CSS sur Wordpress via un thème enfant.

(Wordpress Pauline)
